# FEATURES (상세)

## 인증 / 보안

- Magic Link 비밀번호 없는 로그인
- 미들웨어 세션 동기화 및 보호
- RBAC: 관리자/일반 사용자 DB 역할 기반 접근 제어
- 감사 로그 (로그인/로그아웃/데이터 변경 기록)
- 에러 로깅 시스템 (서버/클라이언트 오류 자동 기록)

## 데이터 수집 & 관리

| 기능          | 설명                                              |
| ------------- | ------------------------------------------------- |
| 업로드        | CSV / XLSX / DAT / MAT, Drag & Drop, 진행률       |
| 청크 업로드   | 대용량 파일 분할 업로드 (init → chunk → finalize) |
| 다중 업로드   | 최대 10개, 50MB 제한, 개별 상태 표시              |
| KPI 자동 계산 | 업로드 시 에너지 효율, 위험도 등 자동 산출        |
| 검색/필터     | 파일명, 날짜, 차량 타입 필터링                    |
| 상세 보기     | 시계열 데이터 테이블 + KPI 요약 카드              |
| 삭제          | 확인 다이얼로그 + 연관 데이터 일괄 삭제           |

## 시각화 / 분석

| 영역        | 제공                                    |
| ----------- | --------------------------------------- |
| 차트 타입   | Line / Bar / Area (Chart.js + Recharts) |
| 상호작용    | 줌 / 팬 / 범례 토글                     |
| 다중 비교   | 여러 측정항목 동시 시리즈               |
| 필터        | 업로드 선택, 날짜 범위 필터             |
| 다운로드    | PNG 이미지 내보내기                     |
| 색상 시스템 | fmai-\* 팔레트 기반 시멘틱 컬러         |

## 관리자 기능

| 영역        | 상태 | 설명                                        |
| ----------- | ---- | ------------------------------------------- |
| 사용자 관리 | 완료 | 사용자 목록, 역할 변경 (user/admin), 삭제   |
| 시스템 통계 | 완료 | 총 사용자, 데이터 포인트, 업로드 수, 활성화 |
| 전체 데이터 | 완료 | 모든 사용자 업로드 파일 조회, 필터, 검색    |
| 감사 로그   | 완료 | 로그인/로그아웃/데이터 변경 이력 조회       |
| 오류 로그   | 완료 | 시스템 에러 목록, 타입별 분류               |

## 성능 / 확장 계획

- 차트 렌더링: 가상화 + 샘플링 예정
- 데이터 인입: 스트리밍(WebSocket) 전환 고려
- Edge Middleware: 인증 빠른 판별
- Incremental Static Regeneration(공개 페이지)
- PWA 지원: 오프라인 기능, 설치 가능

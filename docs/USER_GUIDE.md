## 시작하기

### 시스템 요구사항

| 항목     | 요구사항                                  |
| -------- | ----------------------------------------- |
| 브라우저 | Chrome, Firefox, Safari, Edge (최신 버전) |
| 인터넷   | 안정적인 인터넷 연결 필요                 |
| 화면     | 최소 1024x768 해상도 권장                 |

### 접속 방법

1. 웹 브라우저에서 플랫폼 URL에 접속합니다.
2. 로그인 페이지가 표시됩니다.
3. 이메일 주소를 입력하여 로그인합니다.

## 로그인

### Magic Link 로그인

본 플랫폼은 비밀번호 없이 이메일만으로 로그인하는 Magic Link 방식을 사용합니다.

#### 로그인 절차

1. **이메일 입력**: 로그인 페이지에서 이메일 주소를 입력합니다.
2. **Magic Link 전송**: "Magic Link 전송" 버튼을 클릭합니다.
3. **이메일 확인**: 입력한 이메일 주소로 로그인 링크가 전송됩니다.
4. **링크 클릭**: 이메일에서 "로그인" 링크를 클릭합니다.
5. **자동 로그인**: 대시보드로 자동 이동됩니다.

#### 주의사항

- Magic Link는 **1시간** 동안만 유효합니다.
- 이메일이 도착하지 않으면 스팸 폴더를 확인하세요.
- 링크는 **1회만** 사용 가능합니다.

### 로그아웃

1. 화면 상단 헤더에서 사용자 메뉴를 클릭합니다.
2. "로그아웃" 버튼을 클릭합니다.
3. 로그인 페이지로 이동됩니다.

## 대시보드

로그인 후 가장 먼저 보이는 화면입니다. 전체 데이터 현황을 한눈에 파악할 수 있습니다.

### 주요 구성요소

#### KPI 통계 카드

| 지표         | 설명                             |
| ------------ | -------------------------------- |
| 평균 SOC     | 배터리 충전 상태 평균 (%)        |
| 평균 SOH     | 배터리 건강 상태 평균 (%)        |
| 평균 팩 온도 | 배터리 팩 평균 온도 (°C)         |
| 에너지 효율  | 100km당 에너지 소비량 (Wh/100km) |
| Risk Score   | 주행 위험도 점수                 |

#### 최근 업로드

- 최근에 업로드된 파일 목록을 표시합니다.
- 파일명을 클릭하면 상세 페이지로 이동합니다.

#### 차트 요약

- 주요 지표의 추이를 간략히 시각화합니다.
- "차트 보기" 링크를 클릭하면 상세 차트 페이지로 이동합니다.

## 데이터 업로드

### 접근 방법

1. 상단 메뉴에서 "데이터" > "업로드"를 클릭합니다.
2. 또는 URL `/data/upload`로 직접 접근합니다.

### 지원 파일 형식

| 형식  | 확장자      | 설명                      |
| ----- | ----------- | ------------------------- |
| CSV   | .csv        | 쉼표로 구분된 텍스트 파일 |
| Excel | .xlsx, .xls | Microsoft Excel 파일      |
| DAT   | .dat        | 데이터 파일               |
| MAT   | .mat        | MATLAB 데이터 파일        |

### 파일 업로드 방법

#### 방법 1: 드래그 앤 드롭

1. 업로드 영역에 파일을 끌어다 놓습니다.
2. 파일이 자동으로 업로드 대기 목록에 추가됩니다.

#### 방법 2: 파일 선택

1. 업로드 영역을 클릭합니다.
2. 파일 선택 대화상자에서 파일을 선택합니다.
3. "열기" 버튼을 클릭합니다.

### 업로드 제한사항

| 항목             | 제한                     |
| ---------------- | ------------------------ |
| 최대 파일 크기   | 50MB                     |
| 최대 동시 업로드 | 10개                     |
| 지원 파일 형식   | CSV, XLSX, XLS, DAT, MAT |

### 업로드 진행 상태

| 상태      | 설명                         |
| --------- | ---------------------------- |
| 대기 중   | 업로드 대기 중               |
| 업로드 중 | 파일 전송 중 (진행률 표시)   |
| 처리 중   | 데이터 파싱 및 저장 중       |
| 완료      | 업로드 및 처리 완료          |
| 실패      | 오류 발생 (오류 메시지 확인) |

### 업로드 후 처리

업로드가 완료되면 시스템이 자동으로 다음을 수행합니다:

1. **데이터 파싱**: 파일 내용을 분석합니다.
2. **KPI 계산**: 에너지 효율, 위험도 등 주요 지표를 계산합니다.
3. **저장**: 데이터베이스에 저장합니다.

## 데이터 목록 조회

### 접근 방법

1. 상단 메뉴에서 "데이터" > "목록"을 클릭합니다.
2. 또는 URL `/data/list`로 직접 접근합니다.

### 목록 기능

#### 검색

- 검색창에 파일명을 입력하여 검색합니다.
- 입력 후 Enter 키를 누르거나 검색 버튼을 클릭합니다.

#### 필터링

| 필터      | 설명                             |
| --------- | -------------------------------- |
| 차량 타입 | Tesla, Ionic 등 차량 유형별 필터 |
| 날짜 범위 | 시작일 ~ 종료일 지정             |
| 상태      | 업로드 상태별 필터               |

#### 정렬

- 테이블 헤더를 클릭하여 정렬합니다.
- 파일명, 업로드 날짜, 파일 크기 등으로 정렬 가능합니다.

#### 페이지네이션

- 페이지당 20개 항목이 표시됩니다.
- 하단의 페이지 번호를 클릭하여 이동합니다.

### 테이블 컬럼

| 컬럼        | 설명                                     |
| ----------- | ---------------------------------------- |
| 파일명      | 업로드한 파일 이름 (클릭 시 상세 페이지) |
| 차량 타입   | 데이터의 차량 유형                       |
| 레코드 수   | 파일 내 데이터 포인트 수                 |
| 파일 크기   | 파일 용량                                |
| 업로드 날짜 | 업로드 완료 시간                         |
| 상태        | 처리 상태                                |
| 액션        | 삭제 버튼                                |

### 데이터 삭제

1. 삭제할 데이터 행의 삭제 버튼(휴지통 아이콘)을 클릭합니다.
2. 확인 대화상자가 표시됩니다.
3. "삭제" 버튼을 클릭하여 삭제를 확정합니다.

**주의**: 삭제된 데이터는 복구할 수 없습니다.

## 데이터 상세 보기

### 접근 방법

1. 데이터 목록에서 파일명을 클릭합니다.
2. 또는 URL `/data/[id]`로 직접 접근합니다.

### 페이지 구성

#### 파일 정보

- 파일명, 업로드 날짜, 파일 크기, 레코드 수 등 기본 정보를 표시합니다.

#### KPI 요약 카드

업로드된 데이터에서 계산된 주요 지표를 표시합니다:

| 지표           | 설명                       | 단위     |
| -------------- | -------------------------- | -------- |
| 총 주행거리    | 데이터 기간 중 총 주행거리 | km       |
| 총 에너지 소비 | 사용된 총 에너지           | Wh       |
| 에너지 효율    | 100km당 에너지 소비량      | Wh/100km |
| 평균 SOC       | 평균 충전 상태             | %        |
| 평균 속도      | 평균 주행 속도             | km/h     |
| 평균 팩 온도   | 평균 배터리 온도           | °C       |
| 급가속 비율    | 100km당 급가속 횟수        | 회/100km |
| 급감속 비율    | 100km당 급감속 횟수        | 회/100km |
| 회생 비율      | 회생 제동 에너지 비율      | %        |
| Risk Score     | 종합 위험도 점수           | 점       |

#### 시계열 데이터 테이블

- 원본 데이터를 시간 순서대로 표시합니다.
- 페이지네이션으로 대량 데이터를 탐색합니다.

## 차트 분석

### 접근 방법

1. 상단 메뉴에서 "차트"를 클릭합니다.
2. 또는 URL `/charts`로 직접 접근합니다.

### 차트 기능

#### 데이터 선택

1. 좌측 패널에서 분석할 업로드 데이터를 선택합니다.
2. 여러 데이터를 선택하여 비교할 수 있습니다.

#### 측정 항목 선택

분석할 지표를 체크박스로 선택합니다:

| 항목        | 설명                 |
| ----------- | -------------------- |
| 차량 속도   | 주행 속도 (km/h)     |
| 배터리 전압 | 배터리 팩 전압 (V)   |
| 배터리 전류 | 배터리 전류 (A)      |
| 팩 전력     | 배터리 출력 전력 (W) |
| SOC         | 충전 상태 (%)        |
| 팩 온도     | 배터리 온도 (°C)     |
| 가속도      | 차량 가속도 (m/s²)   |
| Risk Score  | 위험도 점수          |

#### 차트 타입 선택

| 타입 | 용도                    |
| ---- | ----------------------- |
| Line | 시계열 추이 분석 (기본) |
| Bar  | 구간별 비교             |
| Area | 누적 데이터 표현        |

#### 차트 인터랙션

| 기능       | 사용 방법                   |
| ---------- | --------------------------- |
| 줌 인/아웃 | 마우스 휠 스크롤            |
| 팬 (이동)  | 드래그                      |
| 범례 토글  | 범례 항목 클릭              |
| 툴팁       | 데이터 포인트에 마우스 오버 |
| 리셋       | 리셋 버튼 클릭              |

#### 차트 다운로드

1. 차트 우측 상단의 다운로드 버튼을 클릭합니다.
2. PNG 이미지 파일로 저장됩니다.

## 설정

1. 상단 메뉴에서 "설정"을 클릭합니다.

| 옵션   | 설명                 |
| ------ | -------------------- |

## 관리자 기능

관리자 권한이 있는 사용자만 접근할 수 있습니다.

### 접근 방법

1. 상단 메뉴에서 "관리자"를 클릭합니다.
2. 또는 URL `/admin`으로 직접 접근합니다.

### 시스템 통계

| 지표               | 설명                          |
| ------------------ | ----------------------------- |
| 총 사용자 수       | 등록된 전체 사용자 수         |
| 총 데이터 포인트   | 저장된 전체 데이터 레코드 수  |
| 총 파일 업로드     | 업로드된 전체 파일 수         |
| 7일 내 활성 사용자 | 최근 7일간 로그인한 사용자 수 |

### 사용자 관리

#### 사용자 목록

- 모든 등록된 사용자를 조회합니다.
- 이메일, 이름, 역할, 가입일, 마지막 로그인 정보를 확인합니다.

#### 역할 변경

1. 사용자 행의 역할 드롭다운을 클릭합니다.
2. "user" 또는 "admin"을 선택합니다.
3. 변경 사항이 즉시 적용됩니다.

**주의**: 자신의 역할은 변경할 수 없습니다.

#### 사용자 삭제

1. 삭제할 사용자 행의 삭제 버튼을 클릭합니다.
2. 확인 대화상자에서 "삭제"를 클릭합니다.

**주의**: 자신의 계정은 삭제할 수 없습니다.

### 전체 데이터

- 모든 사용자의 업로드 파일을 조회합니다.
- 사용자별, 파일명별 검색이 가능합니다.
- 파일명을 클릭하면 상세 페이지로 이동합니다.

### 감사 로그

시스템 활동 기록을 조회합니다:

| 필드   | 설명                                     |
| ------ | ---------------------------------------- |
| 시간   | 활동 발생 시간                           |
| 사용자 | 활동을 수행한 사용자                     |
| 액션   | LOGIN, LOGOUT, CREATE, UPDATE, DELETE 등 |
| 리소스 | 대상 리소스 유형                         |
| 상세   | 추가 정보                                |

### 오류 로그

시스템 오류 기록을 조회합니다:

| 필드   | 설명                                |
| ------ | ----------------------------------- |
| 시간   | 오류 발생 시간                      |
| 유형   | validation_error, database_error 등 |
| 메시지 | 오류 메시지                         |
| 경로   | 오류 발생 API 경로                  |
| 사용자 | 관련 사용자                         |

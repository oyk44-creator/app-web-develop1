## 로그인 관련

### Q: Magic Link 이메일이 오지 않습니다.

**A:** 다음 사항을 확인해 주세요:

1. **스팸 폴더 확인**: 이메일이 스팸/정크 폴더로 분류되었을 수 있습니다.
2. **이메일 주소 확인**: 입력한 이메일 주소가 정확한지 확인하세요.
3. **잠시 대기**: 이메일 전송에 최대 1-2분이 소요될 수 있습니다.
4. **재전송**: 5분 후에도 도착하지 않으면 다시 시도해 주세요.

### Q: Magic Link를 클릭했는데 "링크가 만료되었습니다" 메시지가 표시됩니다.

**A:** Magic Link는 **1시간** 동안만 유효합니다. 만료된 경우:

1. 로그인 페이지로 돌아갑니다.
2. 이메일 주소를 다시 입력합니다.
3. 새로운 Magic Link를 전송받습니다.

### Q: 로그인 후 바로 로그아웃됩니다.

**A:** 다음 원인을 확인해 주세요:

1. **쿠키 설정**: 브라우저에서 쿠키가 차단되어 있는지 확인하세요.
2. **시크릿 모드**: 시크릿/프라이빗 모드에서는 세션이 유지되지 않을 수 있습니다.
3. **브라우저 캐시**: 브라우저 캐시를 삭제하고 다시 시도해 주세요.

### Q: 여러 기기에서 동시에 로그인할 수 있나요?

**A:** 네, 동일한 계정으로 여러 기기에서 동시에 로그인할 수 있습니다. 각 기기의 세션은 독립적으로 관리됩니다.

## 데이터 업로드 관련

### Q: 파일 업로드가 실패합니다.

**A:** 다음 사항을 확인해 주세요:

| 확인 사항 | 해결 방법                              |
| --------- | -------------------------------------- |
| 파일 크기 | 50MB 이하인지 확인                     |
| 파일 형식 | CSV, XLSX, XLS, DAT, MAT 형식인지 확인 |
| 파일명    | 특수문자 제거 후 재시도                |
| 네트워크  | 인터넷 연결 상태 확인                  |

### Q: 업로드는 완료되었는데 "처리 중" 상태에서 멈춰 있습니다.

**A:** 대용량 파일의 경우 처리에 시간이 걸릴 수 있습니다:

- **10,000 레코드 미만**: 1분 이내
- **10,000 ~ 100,000 레코드**: 2-5분
- **100,000 레코드 이상**: 5-10분

10분 이상 처리 중 상태가 지속되면 새로고침 후 데이터 목록에서 상태를 확인하세요.

### Q: "지원하지 않는 파일 형식입니다" 오류가 발생합니다.

**A:** 파일 확장자가 올바른지 확인하세요:

- 지원 형식: `.csv`, `.xlsx`, `.xls`, `.dat`, `.mat`
- 파일 이름만 변경한 경우 (예: .txt를 .csv로 변경) 실제 형식이 다를 수 있습니다.

### Q: CSV 파일의 데이터가 잘못 인식됩니다.

**A:** CSV 파일 형식을 확인하세요:

1. **인코딩**: UTF-8 인코딩 권장
2. **구분자**: 쉼표(,) 구분 필수
3. **헤더**: 첫 번째 행에 컬럼명 포함
4. **숫자 형식**: 소수점은 점(.) 사용 (쉼표 아님)

### Q: 동시에 여러 파일을 업로드할 수 있나요?

**A:** 네, 최대 **10개** 파일을 동시에 업로드할 수 있습니다. 드래그 앤 드롭으로 여러 파일을 한 번에 선택하거나, Ctrl/Cmd 키를 누른 채 파일을 선택하세요.

## 데이터 조회 관련

### Q: 데이터 목록이 비어 있습니다.

**A:** 다음을 확인하세요:

1. **업로드 확인**: 데이터를 업로드했는지 확인합니다.
2. **필터 해제**: 적용된 필터를 모두 해제합니다.
3. **새로고침**: 페이지를 새로고침합니다.

### Q: 검색 결과가 없습니다.

**A:** 검색 팁:

- 파일명의 일부만 입력해도 검색됩니다.
- 대소문자를 구분하지 않습니다.
- 특수문자는 검색되지 않을 수 있습니다.

### Q: 데이터 상세 페이지에서 일부 데이터만 표시됩니다.

**A:** 대량 데이터는 페이지네이션으로 나누어 표시됩니다:

- 페이지당 100개 레코드가 표시됩니다.
- 하단의 페이지 번호를 클릭하여 다른 데이터를 확인하세요.

### Q: 삭제한 데이터를 복구할 수 있나요?

**A:** 아니요, 삭제된 데이터는 **영구적으로 삭제**되며 복구할 수 없습니다. 삭제 전 확인 대화상자에서 신중하게 결정해 주세요.

## 차트 관련

### Q: 차트가 표시되지 않습니다.

**A:** 다음을 확인하세요:

1. **데이터 선택**: 좌측 패널에서 데이터를 선택했는지 확인합니다.
2. **측정 항목**: 표시할 측정 항목을 하나 이상 선택합니다.
3. **브라우저 호환성**: 최신 브라우저 사용을 권장합니다.

### Q: 차트가 느리게 로딩됩니다.

**A:** 대량 데이터의 경우 로딩 시간이 길어질 수 있습니다:

- 데이터 포인트가 10,000개 이상인 경우 샘플링이 적용됩니다.
- 날짜 범위 필터를 사용하여 데이터 양을 줄여보세요.

### Q: 차트 줌이 작동하지 않습니다.

**A:** 줌 기능 사용 방법:

- **마우스**: 차트 위에서 마우스 휠을 스크롤합니다.
- **터치**: 두 손가락으로 핀치 제스처를 사용합니다.
- **리셋**: 리셋 버튼을 클릭하여 원래 상태로 돌아갑니다.

### Q: 차트를 이미지로 저장하려면 어떻게 하나요?

**A:** 차트 우측 상단의 다운로드 버튼(아래 화살표 아이콘)을 클릭하면 PNG 이미지로 저장됩니다.

### Q: 여러 데이터를 한 차트에서 비교할 수 있나요?

**A:** 네, 좌측 패널에서 여러 업로드 데이터를 선택하면 동일한 차트에 함께 표시됩니다. 각 데이터는 다른 색상으로 구분됩니다.

## 성능 관련

### Q: 페이지 로딩이 느립니다.

**A:** 성능 개선 방법:

1. **브라우저 캐시 삭제**: 오래된 캐시가 문제를 일으킬 수 있습니다.
2. **확장 프로그램 비활성화**: 일부 브라우저 확장 프로그램이 성능에 영향을 줄 수 있습니다.
3. **네트워크 확인**: 인터넷 연결 속도를 확인하세요.
4. **다른 브라우저 시도**: Chrome, Firefox, Edge 등 다른 브라우저를 사용해 보세요.

### Q: 업로드 속도가 느립니다.

**A:** 업로드 속도는 다음에 영향을 받습니다:

- **파일 크기**: 대용량 파일은 시간이 더 걸립니다.
- **인터넷 속도**: 업로드 속도는 네트워크 환경에 따라 다릅니다.
- **서버 부하**: 동시 사용자가 많은 경우 속도가 저하될 수 있습니다.

### Q: 브라우저가 멈추거나 충돌합니다.

**A:** 대량 데이터 처리 시 발생할 수 있습니다:

1. 브라우저 탭을 닫고 다시 접속합니다.
2. 더 작은 데이터 범위로 조회합니다.
3. 브라우저를 최신 버전으로 업데이트합니다.

## 권한 관련

### Q: "접근 권한이 없습니다" 메시지가 표시됩니다.

**A:** 다음 원인을 확인하세요:

| 상황               | 해결 방법            |
| ------------------ | -------------------- |
| 로그인 필요        | 로그인 후 다시 시도  |
| 관리자 전용 페이지 | 관리자에게 권한 요청 |
| 세션 만료          | 다시 로그인          |

### Q: 관리자 페이지에 접근할 수 없습니다.

**A:** 관리자 페이지(`/admin`)는 관리자 역할이 부여된 사용자만 접근할 수 있습니다. 관리자 권한이 필요한 경우 기존 관리자에게 역할 변경을 요청하세요.

### Q: 다른 사용자의 데이터가 보입니다.

**A:** 정상적인 경우 다른 사용자의 데이터는 표시되지 않습니다:

- 관리자 계정인 경우 모든 사용자의 데이터를 볼 수 있습니다.
- 일반 사용자인데 다른 사용자 데이터가 보인다면 관리자에게 문의하세요.

## 기타

### Q: 다크 모드로 변경하려면 어떻게 하나요?

**A:** 두 가지 방법이 있습니다:

1. **설정 페이지**: `/settings`에서 테마를 변경합니다.
2. **푸터 토글**: 화면 하단의 테마 토글 버튼을 클릭합니다.

### Q: 모바일에서 사용할 수 있나요?

**A:** 네, 반응형 디자인으로 모바일 기기에서도 모든 기능을 사용할 수 있습니다. 단, 데스크탑 환경에서 더 나은 사용 경험을 제공합니다.

### Q: 오프라인에서 사용할 수 있나요?

**A:** 제한적으로 가능합니다:

- PWA로 설치된 경우 오프라인 페이지가 표시됩니다.
- 데이터 조회, 업로드 등 서버 통신이 필요한 기능은 온라인에서만 가능합니다.

### Q: 데이터는 어디에 저장되나요?

**A:** 모든 데이터는 Supabase 클라우드 데이터베이스에 안전하게 저장됩니다:

- SSL/TLS 암호화 통신
- Row Level Security(RLS)로 사용자별 데이터 분리
- 정기 백업 수행

### Q: 비밀번호를 변경하려면 어떻게 하나요?

**A:** 본 플랫폼은 비밀번호 없이 Magic Link로 로그인하므로 비밀번호 변경 기능이 없습니다. 매번 이메일로 전송되는 링크로 안전하게 로그인합니다.

### Q: 계정을 삭제하려면 어떻게 하나요?

**A:** 계정 삭제는 관리자에게 요청해야 합니다. 관리자 연락처: support@futuremobility.ai

### Q: 버그를 발견했습니다. 어디에 신고하나요?

**A:** 다음 정보와 함께 support@futuremobility.ai로 문의해 주세요:

- 발생한 문제 상세 설명
- 재현 단계
- 스크린샷 (가능한 경우)
- 브라우저 종류 및 버전
- 발생 시간
